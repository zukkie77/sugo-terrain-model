# SUGO地形モデル製作手順

スポーツランドSUGOの3D地形モデルを、標高データ＋厚紙積層＋レーザーカッターで自作するための手順書です。

---

## 全体工程

| 週 | 内容 |
|----|------|
| Week 1 | 標高データ取得・QGISで等高線生成 |
| Week 2 | SVG整形・枚数確認 |
| Week 3 | FabLab訪問・レーザーカット |
| Week 4 | 積層・接着・塗装 |

---

## Week 1: 標高データ取得 → QGIS で等高線生成

### Day 1-2: 国土地理院からデータ取得

#### 1. アカウント登録

[基盤地図情報ダウンロードサービス](https://fgd.gsi.go.jp/download/menu.jsp) にアクセスし、「利用者登録」から無料登録します。

#### 2. ダウンロード

1. ログイン後「基盤地図情報 数値標高モデル」を選択
2. 「5mメッシュ（DEM5A）」をクリック
3. 地図上でSUGO周辺（宮城県村田町）を選択
   - 検索窓に「スポーツランドSUGO」と入力して中心を確認
4. 該当メッシュにチェックを入れてダウンロード（ZIP形式）

> ダウンロードされるファイル形式: **XML (.xml)**

---

### Day 2-3: QGISのインストールと変換

#### 1. QGISをインストール

[https://qgis.org](https://qgis.org) → 「Download Now」→ Windows版をインストール

#### 2. DEMをGeoTIFFに変換

国土地理院のXMLはそのままでは読み込めないため変換ツールを使います。

- 「基盤地図情報 標高DEMデータ変換ツール」を検索してダウンロード
- ツールにXMLを読み込ませ、GeoTIFF（.tif）に変換

---

### Day 3-5: QGISで等高線を生成

#### 1. GeoTIFFを読み込む

```
QGISを起動
→ レイヤ → レイヤの追加 → ラスタレイヤの追加
→ 変換した .tif ファイルを選択
```

#### 2. 範囲を切り取る（クリップ）

SUGO周辺だけに絞ります。

```
メニュー「ラスタ」→「抽出」→「範囲でクリップ」
→ 地図上でSUGO周辺をドラッグして範囲指定
→ 出力ファイル名を指定して実行
```

#### 3. 等高線を生成

```
メニュー「ラスタ」→「抽出」→「等高線」
→ 入力レイヤ: クリップしたDEM
→ 間隔: 10（メートル）
→ 出力形式: GeoPackage または Shapefile
→ 実行
```

地図上に等高線が線として表示されれば成功です。

#### 4. SVG/DXFでエクスポート

```
メニュー「プロジェクト」→「インポート/エクスポート」→「地図をDXFにエクスポート」
→ ファイル名を指定して保存
```
